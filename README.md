# Naukri-profile-update
Code written > Playwright with python

🚀 Built a Playwright Automation to Keep My Naukri Profile Fresh Automatically

One of the challenges in job portals is maintaining profile freshness and visibility. To solve this, I developed an automation script using Playwright (Python) that automatically updates my Naukri profile at scheduled intervals.

🔹 Automated Login & Navigation
🔹 Dynamic Profile Headline Updates
🔹 Profile Summary Updates
🔹 Success Message Validation
🔹 Error Handling & Logging
🔹 Scheduled Execution Every Hour

The automation validates the response after each update and confirms successful profile changes before completing the execution.
